    <div className="list-item-body">
      The division of the font type will be divided into four parts :
      <ul>
        <li><strong>BOLD</strong> for the title of the in-main article</li>
        <li><strong>SEMIBOLD</strong> for the title on the main page (home page, landingpage menu, list)</li>
        <li><strong>MEDIUM</strong> for text that emphasizes and also clickable</li>
        <li><strong>REGULAR</strong> for reading (paragraphs, descriptions)</li>
      </ul>
    </div>
  </li>
  <li className="list-item">
    <div className="list-item-head">Rule 3</div>
    <div className="list-item-body">

        <li><strong>Header</strong> for the title on the main text </li>
      </ul>
    </div>
  </li>
</ul>
